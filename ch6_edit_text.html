Edit text file

Text editor adalah aplikasi untuk edit text file di OpenBSD. Ianya penting kerana kofigarasi konfigarasi tersebut di simpan dalam text format. Text Editor bukannya Word Processor seperti Libreoffice atau Microsoft office. Di dalam Windows ia adalah aplikasi notepad. Kita boleh memakai dua program untuk edit text file di dalam OpenBSD, vi dan mg.

vi

Saya bukan pengguna vi yang asal pada awalnya saya cuma menggunakan pico editor. vi yang di edar bersama OpenBSD adalah CSRG vi dan bukannya Vim, the editor. vi lebih banyak penguna kerana kebanyakan pegedar Unix atau semacamnya akan menyertakan aplikasi tersebut.

Untuk melancarkan vi, Saya menaip vi dan menekan enter selepas itu saya menaip :version .Disini Kita memberi arahan untuk vi menunjukan apakah versi aplikasi vi yang kita gunakan.

Version 1.79 (10/23/96) The CSRG, University of California, Berkeley.

Begerak dalam vi ia laah dengan h,j,k,l ia itu kiri,ke bawah, ke atas dan ke kanan. i adalah untuk menulis atau a menulis pada hujung barisan. w untuk simpan. Untuk pertolongan kita menggunakan :visuage dan :exusage dan tekan enter. Escape ada lah untuk membatalkan arahan.

Pertolongan lebih lanjut boleh di baca dari man vi

Pada masa awal saya menggunakan vi, saya tidak tahu cara keluar dari applikasi tersebut. Cara keluar ialah menekan :q! untuk keluar sahaja tampa simpan atau :wq! untuk keluar dan simpan

mg

mg adalah text editor seperti GNU zile. Cara permakaian nya adalah sama dengan emacs. Untuk melancarkannya kita menekan mg




-----Mg: *scratch*:          (fundamental)--L1---------------------------------------------------

Pegerakan mg adalah dengan menggunakan arrow keys dan C-s untuk carian.

Untuk melihat pertolong tekan C-h C-h (Control h, Contol h), C-x C-s untuk simpan dan C-x C-c untuk keluar dari Aplikasi
