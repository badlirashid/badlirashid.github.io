Aplikasi Penyunting Teks

Aplikasi penyunting teks (Text editor) adalah penting kerana file konfigarasi konfigarasi di simpan dalam file teks format di dalam OpenBSD. Aplikasi penyunting ini bukan aplikasi Word Processor seperti Libreoffice atau Microsoft office. Penyunting teks tersedia di dalaam Windows adalah notepad. Dua teks editor yang tersedia di dalam OpenBSD adalah semacam vi, vi dan semacam emacs, mg.

vi

Walaupun vi amat popular pada para pengguna, penyunting yang saya mula gunakan adalah pico dan nano. Saya hanya bermula menggunakan vi setelah sekian lama. vi yang di edar bersama OpenBSD adalah CSRG vi dan bukannya Vim yang selalu tersedia di dalam GNU/Linux.

Untuk melancarkan vi, Saya menaip vi dan menekan enter selepas itu saya menaip :version .Disini Kita memberi arahan untuk vi menunjukan apakah versi aplikasi vi yang kita gunakan.

˜
Version 1.79 (10/23/96) The CSRG, University of California, Berkeley.

Versi vi adalah 1.79. Begerak dalam vi kita menggunakan dengan huruf h,j,k,l ia itu ke kiri,ke bawah, ke atas dan ke kanan. Untuk mula menyunting anda perlu tekan i atau tekan a untuk menulis pada hujung barisan. w untuk simpan.

Vi boleh membuka file teks secara lansung dengan vi index.html untuk membuka file teks atau di dalam vi, :open tab untuk menunjukan file file yang ada

˜
˜
+=+=+=+=+=+=+=+=
file1       file2       folder1      folder2     index.html
:open

Untuk pertolongan kita menggunakan :visuage dan :exusage dan tekan enter. Escape ada lah untuk membatalkan arahan. Pertolongan lebih lanjut boleh di perolehi dari man vi

Pada mulanya saya menggunakan vi, saya tidak tahu cara keluar dari applikasi tersebut. Cara keluar ialah menekan :q! untuk keluar sahaja tampa simpan atau :wq! untuk keluar dan simpan.

mg

mg adalah aplikasi penyunting teks yang lebih seperti GNU zile. Aplikasi aplikasi ini semacam GNU Emacs. Untuk melancarkannya kita menaip mg and tekan enter. Berbeza dengan vi, mg tak perlu betukar kepada mod penyuntingan kerana kita terus bekerja dengan teks documen tersebut.




-----Mg: *scratch*:          (fundamental)--L1----------------------------- ----------------------

Arahan mg selalu di gabungkan dengan control dan huruf. C di sini bermaksud control key dan M bermaksud alt atau meta key. Pegerakan mg boleh menggunakan arrow keys atau, M-v untuk ke atas dan C-v untuk ke bawah. Tekan C-x C-s untuk simpan. Untuk mecari ayat dalam teks file kita guna C-s. untuk membuka file guna C-x C-f dan C-x C-c untuk keluar dari mg.

Untuk mengakses sistem pertolongan di dalam mg kita menekan C-h C-h. Kalau di dalam shell kita boleh menggunakan man mg di luar aplikasi.
