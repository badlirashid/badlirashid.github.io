Edit text file

Text editor adalah Aplikasi untuk edit text file di dalam sebuah perisian. Ia nya penting kerana kofigarasi konfigarasi tersebut di simpan dalam text format di dalam OpenBSD. Text Editor bukannya Word Processor seperti Libreoffice atau Microsoft office. Di dalam Windows ia aplikasi notepad. Kita boleh memakai dua program untuk edit text file didalam OpenBSD, vi dan mg.

vi

vi yang di edar bersama OpenBSD adalah BSD vi dan bukannya vim [vim improve]. User vi lebih banyak kerana kebanyakan pegedar Unix atau semacamnya akana menyertakan Aplikasi tersebut.

mg

mg adalah text editor seperti GNU emacs, tetapi lebih ringan.
