cheesysalmon$

Nama cheesysalmon adalah nama sistem dan $ adalah shell prompt. cheesysalmon sedang menunggu arahan dari awak untuk di jalan kan di dalam sistem. Arahan berupa program program didalam file binari ataupun skrip skirp untuk anda lancar kan.

Unix Shell

Secara ringkas, Shell Unix adalah tempat kita berinteraksi dengan sistem OpenBSD. Ia seumpamanya punat punat di kalkulator sedang menunggu petikan anda untuk mengira kiraan anda yang akan di papar di skirin kalkulator itu. Unix Shell yang disediakan adalah Korn Shell atau nama singkatannya adalah ksh. Kalau di dalam perisaian Windows ia adalah 'Command Prompt' atau 'PowerShell'.

Direktori Home Anda

cheesysalmon$ pwd
/home/badli
cheesysalmon$ echo $HOME
/home/badli
cheesysalmon$

Kita mulakan dengan menaip pwd (singkatan Print Working Directory)dan tekan 'Enter' untuk memastikan kita berada di dalam home folder. Ia akan menunjukkan /home/badli di monitor andah.

Satu lagi cari untuk memastikan di mana home folder kita ialah dengan menaip echo $HOME. $HOME ada lah home environment variable. Setiap pengguna sistem ini mempunyaie berlainan home direktori.

Nama host Dan Akaun

cheesysalmon$ whoami
badli
cheesysalmon$ who
janes ttyC0 Feb 23 21:58
badli ttyp0 Feb 23 22:10 (192.168.56.1)
cheesysalmon$ hostname
cheesysalmon.my.domain
cheesysalmon$ uname -a
OpenBSD cheesysalmon.my.domain 7.0 GENERIC#5 amd64

OpenBSD adalah sistem multi pengguna, beberapa pengguna dapat log masuk di dalam sistem tersebut secara serentak. Sebagai contoh saya akses di depan komputer dan ada orang log masuk melauie ssh dari luar.

Di sini saya menggunakan arahan whoami untuk memastikan akaun saya betul. Setelah itu saya mengunakan who akan menunjukkan siapa yang sedang masuk di dalam sistem.

Di sini who memberitahu yang dua pengguna telah masuk ke sistem, saya dan jane.

Arahan hostname melihat kan nama sistem anda sahaja. tetapi uname -a akan memapar kan perisai yang di guna kan dan hostname

File Dan Direktori Yang Disembunyikan

cheesysalmon$ ls
cheesysalmon$ ls -a
. .. .Xdefaults .cshrc .cvsrc .login .mailrc .profile .ssh
cheesysalmon$

Taip ls (singkatan list) untuk melihat kandungan home direktori. Anda tidak nampak apa pun sama ada file atau direktori home kamu.

Sekarang taip ls -a kita akan mendapati ada file dan direktori yang di sembunyikan sebelumnya. -a di sini bermaksud untuk melihat semua file dan direktori walaupun ia di sembunyikan.

File dan direktori di sembunyikan tersebut adalah:-

  • .Xdefaults
  • .cshrc
  • .cvsrc
  • .login
  • .mailrc
  • .profile
  • .ssh

Butiran Penuh File Dan Direktori

cheesysalmon$ ls -al
total 36
drwxr-xr-x 3 badli badli 512 Feb 23 09:22 .
drwxr-xr-x 3 root wheel 512 Feb 23 03:46 ..
-rw-r--r-- 1 badli badli 87 Oct 1 04:00 .Xdefaults
-rw-r--r-- 1 badli badli 769 Oct 1 04:00 .cshrc
-rw-r--r-- 1 badli badli 101 Oct 1 04:00 .cvsrc
-rw-r--r-- 1 badli badli 359 Oct 1 04:00 .login
-rw-r--r-- 1 badli badli 175 Oct 1 04:00 .mailrc
-rw-r--r-- 1 badli badli 212 Oct 1 04:00 .profile
drwx------ 2 badli badli 512 Feb 23 03:44 .ssh

Kita boleh tahu yang mana satu file dan direktori dengan menaip ls -al. -l untuk menunjukkan butiran tersebut.

-rw-r--r-- 1 badli badli 212 Oct 1 04:00 .profile

.profile adalah file kerana di arah permission (-rw-r--r--) tidak ada terdapat huruf d yang menunjukkan ia adalah direktori.

drwx------ 2 badli badli 512 Feb 23 0344 .ssh

.ssh adalah direktori kerana di arah permission (drwx------) terdapat huruf d yang menunjukan ia direktori. Kalau direktori ada huruf x ini membuatkan anda dapat memasuki direktori tersebut.

cheesysalmon$ file .profile
.profile: ASCII text
cheesysalmon$ file .ssh
.ssh: directory

Satu lagi kaedah untuk membeza di antara file dan direktori adalah dengan menggunakan file. Dari contoh program telah membezakan .profile adalah text file dan .ssh adalah folder.

cheesysalmon$ file /sys
/sys: broken symbolic link to 'usr/src/sys'
cheesysalmon$ file /dev/sd0a
/dev/sd1: block special (4/0)
cheesysalmon$ file /bin/ls
/bin/ls: ELF 64-bit LSB shared object, x86-64, version 1

Program file juga dapat memberi tahu beberapa jenis file.

  • /sys - adalah symbolic link, sama seperti shortcut di dalam Windows
  • /dev/sd0a - Ini ada block device untuk cakera keras anda
  • ls - adalah file binari yang kita lacarkan dari shell anda

Semuanya Bermula Dengan /

Di dalam Sistem Unix dan semacamnya permulaan bermula direktori bermula dengan / atau "slash root", di sini semua direktori utama akan dapat dilihat. ia tidak akan membezakan lokasi dengan partitions atau perkakas storage. Seperti pembaca cakera padat ia di akess /mnt/cdrom direktori.
