Konfigurasi OpenBSD

Selesai sahaja pemasang, kita akan membuat beberapa perubahan konfigurasi untuk OpenBSD.

doas pengganti sudo

Aplikasi doas membolehkan kita untuk melancarkan program program yang hanya dapat di lancar dari root dari pengguna biasa. doas mula di perkenalkan di dalam versi OpenBSD 5.8 sebagai pengganti applikasi sudo. Fungsinya mirip seperti Run As Microsofot Windows.

cheesysalmon$ chown www&periodwww index.html
chown: index.html: Operation not permitted
cheesysalmon$ su -
Password:
cheesysalmon# echo 'permit persist keepenv badli' > /etc/doas.conf
cheesysalmon# cat /etc/doas.conf
permit persist keepenv badli
exit
cheesysalmon$ doas chown www.www index.html
doas (badli@cheesysalmon.my.domain) password:
cheesysalmon$ ls -al index&periodhtml
-rw-r--r--  1 www  www  0 Mar  5 12:03 index.html

File doas.conf telah di buat dengan teks untuk membolehkan saya melancarkan program chown sebagai pengguna root.
