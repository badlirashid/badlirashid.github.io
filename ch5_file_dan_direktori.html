File dan Direktori Akses

Sistem Unix Bermula Dengan /

Dalam sistem Unix dan yang semacamnya, permulaan bermula dengan / atau slash root. Di sini semua direktori utama akan bermula dan dapat dilihat. Sistem ini tidak membezakan lokasi dengan berasaskan partitions atau perkakas storage. Seperti pembaca cakera padat ia dapat di akeses dengan /mnt/cdrom direktori.

Berbeza dengan Windows ia bermula dengan C:\ (C Drive) mahupun ia satu fisikal cakera keras atau satu partion. Jikalau ada partion kedua atau pembaca cakera padat ia akaan di namakan sebagai D:\ (D Drive).

Akses

Di dalam Unix Pengguna (Users) dan Kumpulan (Group) adalah cara kita membagi akses kepada file dan direktori.

Akses yang boleh di bagi dalam Unix adalah 3 jenis iaitu:-

  • baca (read)
  • tulis (write)
  • dan lancarkan (execute) program dan skrip

Butiran File

cheesysalmon$ touch filesaya
cheesysalmon$ ls -al filesaya
-rw-r--r-- 1 badli badli 0 Feb 26 13:40 filesaya

Pertama saya memberi arah untuk membuat file kosong bernama filesaya dengan arahan touch.

Menggunakan ls -al, akses filesaya ada lah seperti -rw-r--r--. Akses itu di beri kepada tiga kategori iaitu Pemilik (owner), Kumpulan (group) dan yang lain (others). Akses di baca dari kiri ke kanan, dari numbor dua dan di bagi kepada tiga kategori. Pemilik (owner) di beri baca dan tilis (rw-), sambil kumpulan akses baca sahaja (r--) dan yang lain di beri akses baca sahaja (r--).

File filesaya itu di miliki oleh badli dan group file itu ada lah juga badli

Butiran Direktori

Untuk membuat direktori baru pula, kita gunakan arahan mkdir foldersaya. Apa bila kita beri arahan ls -al butiran directori tersebut seperti berikut:-

cheesysalmon$ mkdir foldersaya
cheesysalmon$ ls -al
drwxr-xr-x 2 badli  badli 512 Feb 26 14:03 foldersaya

d bermaksud direktori dan ini mebezakan ia dengan file. Akses pemilik adalah baca, tulis dan lancarkan rwx, kumpulan ada lah baca dan lancarkan r-x, dan yang lain ada baca dan lancarkan r-x.
x adalah penanda untuk membuat file skrip atau file binari dapat di lancarkan (dalam Windows klik icon dua kali). Tetapi untuk folder pula, ia kita dapat memasuki direktori tersebut.
