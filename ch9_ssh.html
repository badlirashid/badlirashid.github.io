Akses Login Jauh dan Membuat Salinan File

Pengenalan

Program ini membolehkan kita membuat akses kepada Shell Server dari jauh, membuat salinan file dan membuat terowong rangkaian rahsia. Kesemua transmisi data data di rahsiakan. Versi aplikasi bagi OpenBSD 7.0 ketika menulis ini adalah OpenSSH 8.8 dengan LibreSSL 3.4.1. OpenSSH dan LibreSSL ini turut di edarkan dengan sistem operasi Windows 11.

OpenSSH terbahagi kepada dua jenis aplikasi, OpenSSH client atau singkata ssh dan OpenSSH Daemon atau sinkatan sshd. Daemon adalah nama lain bagi Server. OpenSSH adalah program yang di kembangkan majukan oleh Aaron Campbell, Bob Beck, Markus Friedl, Niels Provos, Theo de Raadt dan Dug Song. Ia berasaskan dari aplikasi percuma ssh 1.2.12 yang di keluarkan oleh Tatu Ylonen.

OpenSSH Klient

Ada dua aplikasi asas untuk OpenSSH Klient iaitu ssh dan scp. Saya akan menggunakan OpenBSD dan Windows kerana cara permakaian aplikasi ini sama sahaja

ssh, akses login

Cara menggunakan ssh klient ini ada lah dengan format ssh Nama.Login@Alamat.Server. Nama login adalah nama anda di dalam server tersebut. Alamat Server kita boleh memakai alamat IP atau nama penuh hos.

C:\Users\badli>ssh badli@192.168.144.2
The authenticity of host '192.168.144.2 (192.168.144.2)' can't be established.
ECDSA key fingerprint is SHA256:f/IxY4SmUwJ0i5/sLVhF81QBfAvhGtmSPgivatgWBDE.
Are you sure you want to continue connecting (yes/no/[fingerprint])?

ssh yang di lancar di dalam shell OpenBSD.

Klient ssh akan berhubung dengan server, server itu akan memberi maklumat tentang identifikasinya contoh di atas bewarna biru. Sekiranya Server itu berubah atau identifikasi baru dibuat, identifikasi itu akan berubah. Identifikasi identifikasi ini akan di simpan di dalam folder tersembunyi bernama ssh dan file bernama known_hosts .ssh/known_hosts.

Penerangan boleh di lihat dalam man ssh dan ssh_config.

scp, membuat salinan

scp atau secure copy berasaskan aplikasi rcp atau remote copy.Ia adalah untuk membuat salinan file dari hos ke hos dan data transmisi ini di rashiakan.

Format permakaiannya ialah seperti berikut scp://[user@]host[:port][/path]

C:\Users\badli\Desktop>scp debian.iso badli@192.168.144.2:
badli@192.168.144.2's password:
debian.iso          11%   78MB  13.8MB/s   00:45 ETA

scp sedang membuat salinan file debian.iso ke arah server 192.168.144.2. Format ini sama sahaja mahupun di OpenBSD atau pun Windows. Penerangan lebih lanjut boleh di dapati dalam man scp.

sshd

OpenSSH Daemon atau sshd adalah aplikasi yang mendengar permitaan login daripada klient ssh ataupun scp. Versi OpenSSH yang tersedia hanya menerima SSH protokol versi 2.

Konfigurasi sshd ini terdapat di /etc/ssh/sshd_config. Antara pilihan yang selalu di ubah suai adalah:-

  1. Port - Adalah numbor port ssh konnect. Port 22 ialah selalunya di pakai, kita boleh menggunakan numbor port yang lebih tinggi
  2. listenAddress - Jika ada beberapa alamat, kita boleh set alamat mana yang kita pakai.
  3. PermitRootLogin - Jika root boleh login secara langsung atau tidak. Default is no.

Dokumentasi lebih lanjut boleh di lihat dalam man sshd dan sshd_config.
